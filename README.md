# Mario Percivaldi — Claude Code Plugins

Personal marketplace of Claude Code plugins.

## Add this marketplace

```shell
/plugin marketplace add marioPercivaldi/claude-plugins
```

## Plugins

### docmd

Convert file attachments (PDF, DOCX, PPTX, XLSX, images, audio, CSV, JSON, XML, HTML, EPUB) to Markdown using Microsoft MarkItDown, reducing token usage by up to 80%.

```shell
/plugin install docmd@mario-plugins
```

→ [Repository](https://github.com/marioPercivaldi/docmd)
